redCore Linux Project

redCore Linux is a distribution based on Kogaion Linux and Gentoo Linux. redCore bring the latest technologies and better support for modern hardware technologies.

Minimum system requirements:
Processor: Single – Dual Core 64Bit
RAM Memory: 1 – 2 GB
Disk Space (HDD): 10 – 15 GB

Recommended system requirements:
Processor: Quad Core 64bit
RAM Memory: 8 GB
Disk Space (SSD): 20 GB
